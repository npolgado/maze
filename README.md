# MAZE GAME

Project/Game idea from Eric Nicholls 
- 2D top down maze game
- Player has to go through `stages` or phases on the maze, with `gates` or continutation to other parts of the maze, which expresses a unique story per playthrough
- Braid maze that is custom, least k-length loops, multiple paths to exit.
